Linear Search

Linear search is one of the most simple searching algorithms. Put simply, it goes through every item in the list and checks if that is the value it's looking for. If not, go to the next value, and keep doing this until either the item is found or the end of the list. Linear search has a time complexity of O(n) and Ω(1).
